# AI Tech Company CEO Simulator

## 프로젝트 개요

AI Tech Company CEO Simulator는 C언어를 이용하여 제작한 텍스트 기반 경영 시뮬레이션 게임입니다.

플레이어는 AI 및 IT 분야의 빅테크 기업 CEO가 되어 회사를 운영하게 됩니다. 게임은 총 10분기 동안 진행되며, 매 분기마다 다양한 경영 전략을 선택하여 회사를 성장시켜야 합니다.

회사의 자금, 사용자 수, 주가, 직원 수, 평판 등의 지표가 존재하며 플레이어의 선택과 랜덤 이벤트에 따라 회사의 성과가 달라집니다. 목표는 파산하지 않고 10분기 동안 회사를 성공적으로 운영하여 높은 등급을 받는 것입니다.

---

## 게임 목표

* 회사를 파산시키지 않고 10분기 동안 운영한다.
* 사용자 수를 확보하여 수익을 창출한다.
* 주가와 평판을 높여 기업 가치를 성장시킨다.
* 최종적으로 가장 높은 등급(S)을 획득한다.

---

## 주요 기능

### 1. 회사 상태 관리

매 분기마다 현재 회사의 상태를 확인할 수 있습니다.

출력 정보:

* 자금(Money)
* 사용자 수(Users)
* 주가(Stock Price)
* 직원 수(Employees)
* 평판(Reputation)
* 시장 점유율(Market Share)

시장 점유율은 다음과 같이 계산됩니다.

시장 점유율 = 사용자 수 ÷ 100000 × 100

---

### 2. 경영 의사결정 시스템

플레이어는 매 분기마다 다음 다섯 가지 행동 중 하나를 선택할 수 있습니다.

#### R&D 투자

연구개발에 투자하여 기업의 기술력을 향상시킵니다.

효과:

* 자금 감소
* 사용자 증가
* 주가 증가
* 직원 증가
* 평판 증가

#### 마케팅

광고와 홍보를 통해 사용자를 확보합니다.

효과:

* 자금 감소
* 사용자 크게 증가
* 주가 증가
* 평판 감소

#### 채용

회사의 인력을 확충합니다.

효과:

* 자금 감소
* 주가 증가
* 직원 증가
* 평판 증가

#### 구조조정

직원 수를 줄여 비용을 절감합니다.

효과:

* 자금 증가
* 사용자 감소
* 주가 감소
* 직원 감소
* 평판 감소

#### 신제품 출시

새로운 제품을 출시하여 성장을 도모합니다.

효과:

* 자금 감소
* 사용자 증가
* 주가 크게 증가
* 평판 증가

---

### 3. 랜덤 이벤트 시스템

매 분기마다 랜덤 이벤트가 발생하여 회사 상태에 영향을 줍니다.

이벤트 종류:

* 앱 대박 입소문
* 데이터 유출 사고
* 투자 유치 성공
* 서버 장애 발생
* 핵심 인재 채용
* 정부 규제 강화

각 이벤트는 회사의 자금, 사용자 수, 주가, 직원 수, 평판에 다양한 영향을 미칩니다.

---

### 4. 분기 정산 시스템

매 분기 종료 시 회사의 수익과 비용을 계산합니다.

#### 수익 계산

사용자 수를 기반으로 수익을 계산합니다.

수익 = 사용자 수 ÷ 10

#### 비용 계산

직원 수를 기반으로 인건비를 계산합니다.

비용 = 직원 수 × 10

#### 순수익 계산

순수익 = 수익 − 비용

정산 결과는 회사 자금에 반영됩니다.

---

### 5. 파산 시스템

회사의 자금이 0 이하가 되면 파산합니다.

파산 시:

* 게임 즉시 종료
* 최종 결과 출력
* 재시작 여부 선택 가능

---

### 6. 최종 평가 시스템

10분기가 종료되면 회사의 성과를 종합적으로 평가합니다.

평가 요소:

* 자금
* 사용자 수
* 주가
* 직원 수
* 평판

각 요소를 합산하여 최종 점수를 계산합니다.

---

## 등급 체계

| 등급 | 조건        |
| -- | --------- |
| S  | 18000점 이상 |
| A  | 15000점 이상 |
| B  | 12000점 이상 |
| C  | 8000점 이상  |
| D  | 5000점 이상  |
| F  | 5000점 미만  |

등급에 따라 서로 다른 결과 메시지가 출력됩니다.

예시:

* S : 세계적인 빅테크 기업으로 성장했습니다!
* A : 유니콘 기업 등극!
* B : 안정적인 성장 기업이 되었습니다.
* C : 평범한 중견 기업 수준입니다.
* D : 간신히 생존에 성공했습니다.
* F : 회사 파산. CEO에서 해고되었습니다.
